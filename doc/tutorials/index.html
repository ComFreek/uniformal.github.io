Tutorials

This folder collects various tutorials given for individual aspects of using MMT.

  • OMDoc/MMT for Mathematicians is a general tutorial on working with MMT content. It covers creating new archives, the MMT language, using LF, working with jEdit, building archives, browsing the MMT server.

  • Language Design in MMT is the first part of a tutorial given at CICM 2016 about how to use the MMT IDE to rapidly prototype formal systems.

  • Getting Started as an MMT API user is a quick walkthrough on how to set up a Scala project for MMT API usage showing a simple example on how to list all theories of an archive.

  • Application Development with MMT is the second part of a tutorial given at CICM 2016 about how to build applications on top of the MMT library.

  • Writing MMT documents with jEdit is a general tutorial about using the MMT IDE (possibly outdated)

MMT is developed and maintained mostly by Florian Rabe.
